Модуль вывода содержимого корзины в PDF.
Автор: Gennady Telegin <support@itxd.ru>

Модуль добавляет ссылку в корзину для скачивания её содержимого в PDF.

Внешний вид PDF редактируется также, как и любой другой файл темы.
Файл шаблона: catalog/view/theme/default/module/pdfcart.tpl

Для установки скопируйте содержимое каталога upload на сервер.

Требуется VQMod (для добавления ссылок в корзину).

Совместим с модулем Simple.
